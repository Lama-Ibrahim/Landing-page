# موقع زاينا - Landing Page

موقع منفصل تماماً عن تطبيق React Native، مصمم بناءً على شعار زاينا.

## المميزات

- تصميم بسيط وأنيق يعكس هوية العلامة التجارية
- ألوان مستوحاة من الشعار (بنفسجي فاتح وخلفية بيج)
- متجاوب بالكامل (يعمل على جميع الأجهزة)
- دعم كامل للغة العربية (RTL)
- أداء سريع (HTML/CSS/JS خفيف)

## البنية

```
website/
├── index.html      # الصفحة الرئيسية
├── styles.css      # ملف التنسيقات
├── script.js       # ملف JavaScript
└── README.md       # هذا الملف
```

## كيفية الاستخدام

### فتح الموقع محلياً

1. افتح ملف `index.html` مباشرة في المتصفح
2. أو استخدم خادم محلي:

```bash
# باستخدام Python
python -m http.server 8000

# باستخدام Node.js (http-server)
npx http-server

# باستخدام PHP
php -S localhost:8000
```

ثم افتح المتصفح على `http://localhost:8000`

### النشر

يمكنك رفع محتويات مجلد `website/` إلى أي استضافة ويب:
- Netlify
- Vercel
- GitHub Pages
- أي استضافة عادية

## التخصيص

### الألوان

يمكنك تعديل الألوان من ملف `styles.css`:

```css
:root {
    --brand-purple: #9884b4;  /* اللون البنفسجي الرئيسي */
    --taupe-bg: #e8e0d6;      /* لون الخلفية */
    --dark-purple: #6b5a84;   /* بنفسجي داكن */
}
```

### المحتوى

عدّل المحتوى مباشرة من ملف `index.html`

## المتطلبات

- متصفح حديث يدعم CSS Grid و Flexbox
- لا يحتاج أي مكتبات خارجية (باستثناء خطوط Google Fonts)
