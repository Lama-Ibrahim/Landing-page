# ملفات الشعار - zaina Logo

تم إنشاء ملفات SVG للشعار بجودات وألوان مختلفة.

## الملفات المتوفرة:

### 1. `logo.svg`
- الشعار الأساسي باللون البنفسجي (#9884b4)
- الحجم: 160x160 بكسل
- للاستخدام على الخلفيات الفاتحة

### 2. `logo-white.svg`
- نفس الشعار بلون أبيض (#ffffff)
- الحجم: 160x160 بكسل
- للاستخدام على الخلفيات الداكنة (مثل Footer)

### 3. `logo-with-text.svg`
- الشعار مع كلمة "zaina" تحته
- الحجم: 200x160 بكسل
- للاستخدام في العروض التقديمية أو المواد التسويقية

## كيفية الاستخدام:

### في HTML:
```html
<img src="logo.svg" alt="zaina logo" />
```

### في CSS:
```css
background-image: url('logo.svg');
```

### في React/Next.js:
```jsx
import Logo from './logo.svg';

<img src={Logo} alt="zaina logo" />
```

### في InDesign/Illustrator:
- افتح الملف مباشرة في Adobe Illustrator
- يمكنك تحريره وتعديله حسب الحاجة
- احفظه كـ PNG أو PDF حسب الاستخدام

## تخصيص الألوان:

يمكنك تغيير لون الشعار بتعديل قيمة `stroke` في ملف SVG:
- البنفسجي: `#9884b4`
- الأبيض: `#ffffff`
- أي لون آخر: استبدل قيمة hex
